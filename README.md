
 ## Process Management (Operating System)
This project for CO2 Process Management chapter in Operating System.

## Topics Included
- What is a Process? 
- Process States (New, Ready, Running, Waiting, Terminated)
- Process Control Block (PCB) 
- Process Scheduling
- Context Switching 
- Inter Process Communication (IPC) 
- Threads Concept
- Types of Threads (User-level, Kernel-level)
- Thread Models (Many-to-One, One-to-One, Many-to-Many)
- Thread Libraries
- Thread Issues (Race condition, Deadlock, Starvation, Synchronization)
